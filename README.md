# Чтобы запустить проект необходимо:

### 1. Cклонить его с помощью команды
### `git clone https://github.com/daria-bond/test_bfg.git`

### 2. Установить node modules с помощью команды
### `npm install`

### 3. Запустить проект с помощью команды
### `npm run start`
